# Conventions

## Naming conventions

| Identifier | Naming convention | Example |
| :--- | :--- | :--- |
| constant  | `camelCase` | `pi` |
| function  | `camelCase` | `show` |
| tag  | `camelCase` | `true` |
| type  | `PascalCase` | `Int` |
| tagged union  | `PascalCase` | `Color` |
| interface  | `PascalCase` | `Comparable` |
